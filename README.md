# Sistema de Información Web para la Gestión, Control y Trazabilidad Documental

## Seguro Social Universitario de Tarija

Es un proyecto de la Universidad Domingo Savio

### Descripción
Sistema web desarrollado para optimizar la gestión documental del Seguro Social Universitario de Tarija, permitiendo la clasificación estructurada, localización eficiente y registro sistemático de movimientos de documentos físicos.

### Tecnologías
- **Backend**: ASP.NET Core 8.0 (C#) con arquitectura MVC
- **Frontend**: Flutter (Web y Móvil)
- **Base de Datos**: PostgreSQL
- **Metodología**: Scrum

### Estructura del Proyecto
```
Sistema_info_web_gestion/
├── backend/              # API RESTful en ASP.NET Core
├── frontend/             # Aplicación Flutter
├── database/             # Scripts SQL de PostgreSQL
└── docs/                 # Documentación
```

### Funcionalidades Principales
1. Registro y clasificación de documentos con metadatos
2. Control de entrada y salida de documentos
3. Trazabilidad documental completa
4. Búsqueda avanzada de documentos
5. Generación de códigos QR para identificación
6. Reportes y monitoreo automatizado

### Requisitos
- .NET 8.0 SDK
- Flutter SDK 3.0+
- PostgreSQL 14+
- Visual Studio 2022 o VS Code
